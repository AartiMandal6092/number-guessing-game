# 🎯 Number Guessing Game

A simple Python console game where the computer selects a random number between 1 and 100, and the user has to guess it.

---

## 🚀 Features
- Random number generation
- Tracks the number of attempts
- Gives hints (Too high / Too low)
- Simple interactive CLI game

---

## 🛠️ Tech Stack
- Python
